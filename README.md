# Birthday Card for Rylee Rae Stiles

An elegant birthday card with a soft reveal animation, photo gallery, and personal note.

## How to use

1. Open `index.html` in a web browser.
2. Wait for the intro to finish.
3. Click `Open Your Gift` to open the gift.
4. The gift opens a photo gallery and a personal note. Select any photo to view it larger.

To change photos, replace the image file names and captions in `index.html`. The current gallery uses four images stored beside `index.html`.

## Files

- `index.html` — page structure and message.
- `styles.minimal.css` — active styling for the flip-card, responsive layout, and photo carousel.
- `styles.css` — an older alternate stylesheet that is not used by the page.
- `script.js` — intro, gift reveal, modal and image interactions.
